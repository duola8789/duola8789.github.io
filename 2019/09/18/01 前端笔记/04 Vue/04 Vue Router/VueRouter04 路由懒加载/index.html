Vue router 路由懒加载学习笔记。

路由懒加载

路由懒加载解决的是一次加载的JS包过大的问题，将整个的JS包按照不同的路由对应的组件分割成为不同的代码块，路由访问的时候再加载对应的组件

这样的好处就是首次加载时的速度很加快，但是路由切换的过程，由于组件本身也需要通过网络请求获取，所以性能会降低

实现需要借助Vue的异步组件和Webpack的代码分割功能
